## 三句话 · 规则 + 相对数据 + 少量 LLM（PROFILE-THREE-SENT-01 · P0）

> 详文 → [`profile-investment-requirements-report-spec.md`](../../requirement/docs/samples/profile-investment-requirements-report-spec.md) §3.1

**流水线**：`buildProfileReportMarkdown`（模板 + 槽位 + 规则草稿）→ `refineProfileDraftReport`（LLM 润色 **三句话 + §6**；失败则用规则草稿）

### 相对数据（有才算）

| 键 | 公式 / 条件 | 示例话术 |
|----|-------------|----------|
| `principal_pct_of_assets` | 本组已有 ÷ 可投资金融资产 | 本组 **12 万** 约占可投资资产 **24%** |
| `monthly_pct_of_investable` | 本组月投 ÷ 月可投 | 月投 **3,000** 约占月可投 **86%** |
| `months_to_full_deploy` | 定投且月投 >0 时粗算 | 按当前节奏约 **24 个月** 投完 |
| `debt_payment_ratio` | 月还贷 ÷ 月到手 | 还贷占月到手约 **36%** |
| `surplus_after_group` | 月到手 − 生活 − 还贷 − 本组月投 | 本组外月结余约 **500 元** |
| `risk_coherence` | 见下表 | 三者 **大致匹配** |

### 风险自洽标签（② 用 · 规则判定）

| 条件 | `risk_coherence` |
|------|------------------|
| 稳健/保守 + 回撤 ≤15% + 预期 ≤6% | **大致匹配** |
| 平衡 + 回撤 ≤20% + 预期 ≤10% | **大致匹配** |
| 进取/积极 或 预期偏高相对回撤 | **偏进取，需留意波动** |
| 保守 + 预期 ≥8% 或 回撤 ≥25% | **边界偏紧，建议后续配置讨论对齐** |

### 规则草稿骨架（LLM 前 · ①②③ 标签固定）

```markdown
> **① 资金性质：** 针对「{场景}」的{角色句}；{principal_pct 一句}。
> **② 风险偏好：** 自评 **{risk_tolerance}**，最多约 **{max_drawdown}**，预期年化约 **{expected_return}**——{risk_coherence 句}。
> **③ 执行节奏：** 本组 **已有 {principal}**，每月 **再投 {monthly}**（{deploy_mode}）；{monthly_pct 或 months_to_full_deploy 一句}。
```

## §6 对您需求的理解（综合解读 · P0）

> **定位**：对标参考 `reference-project/.../徐美丽_客户画像.md`（历史文件名）的「客户特征分析」，但 **更克制**——只做 **需求含义** 与 **约束自洽性**，不做市场判断。

**必写维度（≥3 条，依场景取舍）**

| 维度 | 写什么 |
|------|--------|
| **本组角色** | 这组钱在整体财务里扮演什么（应急 / 长期 / 阶段性大额等） |
| **风险自洽** | 风险偏好、回撤上限、收益预期是否互相匹配；需提醒的边界 |
| **资金与执行** | 本组金额 vs 客户信息层结余；定投/一次性是否与能力匹配 |
| **流动性** | 动用计划对 **持有与波动** 的含义（仍 **不** 点名产品） |

**§6 禁止**：`web_search`、宏观观点、`fund_code`、大类比例、买卖建议、复读 §3–§5 整表、**Tab/流程指引**（如「下一步去资产配置」）。

## 本组投资目标

本组为 **退休养老**——退休生活超长期储备，退休前原则上不动。

## 本组投资偏好与边界

- **风险偏好**：稳健  
- **最多能接受回撤**：约 -15%  
- **期望年化收益**：约 4%–5%  
- **什么时候可能动用**：退休前原则上不动  
- **投入方式**：每月定投，约 24 个月投完

## 对您需求的理解

> 以下基于上文 **您已确认** 的信息归纳…

**1. 本组角色** 距退休约 **25 年**，时间维度 **充裕**，可承受 **权益类的小比例** 长期增值，但不宜按「短期理财」设计。

**3. 资金与执行** 本组月投 **3,000 元** 占客户信息层月可投 **3,500 元** 的大部分；需留意 **还贷款 6,500 元** 后家庭现金流 **偏紧**，**不宜** 在本组外 **重复占用** 同一笔结余。
