## 9.3 我的自选

### 模块说明

| 项 | 说明 |
|----|------|
| **做什么** | 用户 **常看基金** 列表；一键 **AI 解析**；可随时增删（含预置三只） |
| **入口** | Tab **基金** → 主区 **我的自选**（§1.2.5） |
| **完成标志** | 增删成功；AI 解析跳转 [analysis §9.1.2](./09-fund-analysis.md) |
| **不做** | 报告正文、PDF 维护（→ §9.1 / §9.2） |
| **编码锚点** | `fund_watchlist` §9.3.8 · WL-01～WL-04 |

Hub 对照与三支柱关系 → [§9.0](./09-fund.md)。**与持仓/报告关系**见 §9.3.8。

### 9.3.1 布局（Figma）

| fund 模式 | 主区 Tab |
|-----------|----------|
| **模式 A**（无草稿） | **对话** \| **我的自选** |
| **模式 B**（有草稿） | **报告预览** \| **我的自选** |

| 项 | 规格 |
|----|------|
| 列表 | 简称 + 代码；**AI 解析** / **删除自选** |
| 顶栏 | 搜索添加（P0） |

| 排序 | `added_at` DESC；**不做**拖拽 |

### 9.3.2 交互

| 操作 | 行为 |
|------|------|
| **添加** | ① Tab 顶栏搜索 → `POST /api/fund-watchlist` ② 对话 **加入自选确认卡**（§9.3.2a） ③ 报告发布后 **建议卡**（§9.3.2b · WL-04） |
| **删除自选** | 列表行 **「删除自选」**（线框 **[ 删除 ]**）→ **二次确认** → `DELETE /api/fund-watchlist/{fund_code}` → 列表刷新；**含预置三只均可删**；**不删**已发布报告 |
| **AI 解析** | 切 **对话** Tab → **注入用户消息**「请就 {code} 出具完整基金解读报告」→ `fund_full_report`（[analysis §9.1.2](./09-fund-analysis.md)） |

### 9.3.3 Command（自选域 · fund Tab · `/` 补全）

> **共有** `/` 机制 → [shared §5.3.9a](./05-chat-shared.md) · `usage_pages.fund` · 自选分组。

| Command | 对客说明 | 读写 |
|---------|----------|------|
| `fund_search` | 搜索可添加的基金 | 读 |
| `fund_watchlist_add` | 加入我的自选 | 写 |
| `fund_watchlist_remove` | 从我的自选移除 | 写 |

### 9.3.4 API（P0）

| 方法 | 路径 | 说明 |
|------|------|------|
| `GET` | `/api/fund-watchlist` | `added_at` DESC |
| `POST` | `/api/fund-watchlist` | `{ fund_code }`；`fund_lookup` 补全名称；重复 → **409** |
| `DELETE` | `/api/fund-watchlist/{fund_code}` | 删一行 |
| `GET` | `/api/funds/search?q=` | 顶栏搜索 |

### 9.3.6 边界行为

| 场景 | 行为 |
|------|------|
| **删除自选（踢出）** | 用户点 **删除自选** → 确认 → 删 `fund_watchlist` 行；**默认 3 只同样可删**；删光后展示空状态（§9.3.1） |
| **重复加入** | 「已在您的自选列表中」 |
| **发布后不在自选** | WL-04 建议卡 |
| **对话内加入** | 确认卡 → `fund_watchlist_add` |

解析侧边界见 [analysis §9.5](./09-fund-analysis.md)。

### 9.3.8 字段规格（Supabase）

#### 9.3.8a `fund_watchlist`

| 中文含义 | 字段名称 | 字段类型 | 字段长度 | 是否必填 | 字段校验 | 值的相关说明 |
|----------|----------|----------|----------|----------|----------|--------------|
| 主键 | `id` | uuid PK | uuid | 系统 | — | — |
| 基金代码 | `fund_code` | text **UNIQUE** | — | 是 | 国内公募 | 单实例不可重复 |
| 基金简称 | `fund_name` | text | — | 是 | 写入时 L0 带出 | 可缓存 |
| 加入时间 | `added_at` | timestamptz | — | 系统 | — | 侧栏默认排序 |
| 最近解读时间 | `last_analysis_at` | timestamptz? | — | 否 | — | 发布 fund 报告时回写 |

**与持仓、报告**：与 `holdings_versions` **独立**；与 `report_index`（fund）**无外键**，仅同 `fund_code` UI 关联；删自选 **不删** 已发布报告。

**默认已添加自选（MVP · WL-01 · DEMO-ABCDEF-01）**：migration/seed 空表时 INSERT **6 行** — 019305 / 017704 / 110020 / 206007 / 519772 / 518880（archetype **A～F 各一**，见 [Hub §9.0.1](./09-fund.md)）；用户可删可增；**不代表已持仓**。
