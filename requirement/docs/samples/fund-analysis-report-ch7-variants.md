## 组合 1 · 无投资需求 · 无其它自选（最小完整 §4）

```markdown
## 这只基金适合我吗？

**本章回答：** 不同风险偏好下的 **匹配度**（非买卖建议）。

### 通用匹配对照

| 如果您是… | 匹配度 | 原因 |
|-----------|--------|------|
| **偏稳健**，希望几乎不亏、随时可取 | 偏低 | 海外权益 + 汇率，不适合当「活期理财」 |
| **平衡型**，想分散一些到海外 | 中等 | 可作 **全球配置的一部分**，建议控制占比 |
| **偏进取**，认可长期持有美国大盘 | 较匹配 | 与中长期海外权益目标较为一致 |

若您希望结合 **投资期限、回撤承受、资金用途** 等个人约束做更细对照，可先在 **需求梳理** Tab 完成一组投资需求并发布报告；在 **基金 · 我的自选** 中加入不同类型基金后，下次解读也会自动带上 **对照小节**。
```
