## 2. LLM 导语（三句话 + 第二·三章）

> **生成时机**：`fund.rpt.draft` 模板拼装 → `fund.rpt.draft.compose` LLM 批量写入。  
> **防重复**：三句话 **定性为主**；**业绩数字主战场**在第二章导语 + **`l0_summary` 数字块** + 产品表。

### 2.1 三句话读懂这只基金

固定 **3 句、3 主题**；LLM 只填槽，不换序、不加第 4 句。

| 句序 | 主题 | 写什么 | 主要输入 |
|------|------|--------|----------|
| **① 产品定位** | 这是什么、投什么 | L0 `fund_type` 压缩后的类型标签 + L1 投资目标/策略压缩 1 句 | L0 + L1 `prospectus` |
| **② 风险画像** | 波动特征与风险 | L1 `risk_level` + 该类型的特征风险（如权益高仓位、海外汇率、被动跟踪误差、固收信用等） | L1 风险揭示 + archetype |
| **③ 适配提示** | 谁适合、持有多久 | 建议持有期限 + 适合人群/场景描述 | archetype |

**字数（已拍板 · 放宽一倍）：**

| 项 | 限制 |
|----|------|
| 每句 | **40～90 汉字**（含标点；不含 `①②③` 标签） |
| 三句合计 | **≤240 字** |
| 格式 | 3 行 blockquote；每行 1 句；禁止 bullet |
| 禁止 | 内部词、收益承诺、具体申购费率 |

**LLM 失败**：TPL 按 archetype 三槽 fallback（产品定位 / 风险画像 / 适配提示），不凑假数。

### 2.2 第二章「这只基金赚不赚钱」

```markdown
## 这只基金赚不赚钱

（LLM 开篇 1 段普通段落，**禁止「本章回答」**）

{l0_summary 数字块 — **保留，不改写**}
```

| 项 | 规范 |
|----|------|
| **开篇** | LLM · **1 段 · 120～200 字** · 普通段落（非 blockquote） |
| **结构** | 结论 → 近一年收益/回撤/基准超额（**须用输入 JSON 中的 L0 数字**）→ 非承诺限定 |
| **输入** | `return_1y_pct`、`max_drawdown_1y_pct`、`benchmark_name`、`benchmark_return_1y_pct`、`excess_return_1y_pct`、`archetype` |
| **l0_summary** | 模板直出，compose **不得删除或改写** |

**基准三档**（写在导语中，有超额数字时）：

| 条件 | 填空句 |
|------|--------|
| 超额 **> +0.5%** | 近一年 **跑赢** 业绩基准约 **{excess}%** |
| 超额 **< −0.5%** | 近一年 **跑输** 业绩基准约 **{abs(excess)}%** |
| **±0.5%** 内 | 与业绩基准 **大致接近** |
| 仅有基准名 | 业绩比较基准为 **{benchmark_name}** |

### 2.3 第三章「是否适合长期持有」

```markdown
## 是否适合长期持有

（LLM 开篇 1 段普通段落，**禁止「本章回答」**）

### 风险揭示摘要
（L1 摘录 + compose 润色）
```

| 项 | 规范 |
|----|------|
| **开篇** | LLM · **1 段 · 120～200 字** |
| **结构** | 适合谁/不适合谁 → 建议持有期 → 长期主要风险 |
| **输入** | L1 投资目标/策略/申赎规则、`risk_level`、`fund_type`、`archetype` |
| **禁止** | 业绩数字（`X%` 涨跌）；「一定适合长期持有」 |

### 2.4 Verify（导语相关）

| 检查 | 规则 |
|------|------|
| 三句话 | 存在 `## 三句话`；3 行；合计 ≤240 字 |
| 三句话 ① | 含类型或投资目标语义 |
| 第二章 | 有开篇段 + **其下仍有 l0_summary**；**禁止「本章回答」** |
| 第三章 | 有开篇段；正文无业绩 `%`；**禁止「本章回答」** |
| 内部词 | 全文禁止 L0/L1/Tushare/AKShare 等 |
