# 鹏华消费优选混合型证券投资基金 · 基金产品资料概要

编制说明：本文件为演示知识库种子，内容依据鹏华基金管理有限公司公开信息整理。

## 一、产品概况

| 项目 | 内容 |
|------|------|
| 基金简称 | 鹏华消费优选混合 |
| 基金代码 | 206007 |
| 基金类型 | 混合型 · 偏股 · 消费主题 |
| 基金管理人 | 鹏华基金管理有限公司 |
| 基金托管人 | 中国银行股份有限公司 |
| 成立日期 | 2010年8月20日 |
| 运作方式 | 开放式 |
| 基金经理 | 陈璇（演示种子口径） |

## 二、投资目标

在合理控制投资风险的前提下，精选消费行业优质上市公司，追求基金资产的长期稳健增值。

## 三、投资范围

本基金主要投资于具有消费行业特征的 A 股上市公司股票，包括：

- 食品饮料、家电、商贸零售、社会服务、传媒等消费相关板块
- 债券、货币市场工具及法律法规允许的其他金融工具（比例依合同）

**投资限制（摘要）**：

- 股票投资比例通常为基金资产的 **60%—95%**
- 不得投资 ST、*ST 股票（除已纳入指数成份等法定例外）

## 四、业绩比较基准

中证消费指数收益率 × 60% + 中证综合债指数收益率 × 40%

## 五、风险收益特征

本基金属于 **混合型基金**，预期风险和预期收益高于债券型基金和货币市场基金，低于股票型基金。

| 风险等级 | 说明 |
|----------|------|
| 典型风险等级 | **R4 中高风险**（以销售机构评定为准） |
| 主要风险 | 市场风险、行业集中风险、流动性风险、基金经理变更风险 |

## 六、费率结构

| 费用类型 | 费率 |
|----------|------|
| 管理费 | **1.20%** / 年 |
| 托管费 | **0.20%** / 年 |
| 申购费（前端） | 0.15%（100 万以下示例档） |
| 赎回费 | 持有不满 7 天 1.50%；7 天—365 天 0.50%；365 天以上 0.00% |

## 七、分红政策

在符合分红条件时，基金管理人可根据基金运作情况分配收益；是否分红、分红次数与比例不保证。
