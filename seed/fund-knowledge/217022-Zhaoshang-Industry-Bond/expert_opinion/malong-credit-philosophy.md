# 马龙信用债投资方法论

**观点类型**：基金经理投资框架研究  
**适用基金**：217022 招商产业债券A

## 一、信用研究三级体系

马龙（招商基金固收投资总监）建立了一套独特的信用研究三级体系，贯穿产业债投资的全部流程：

| 研究层级 | 核心内容 | 关键指标 | 淘汰标准 |
|----------|---------|---------|---------|
| 第一层：行业景气 | 行业供需格局、政策环境 | 行业ROE、现金流覆盖率 | 强周期行业一票否决 |
| 第二层：企业财务 | 盈利能力、偿债能力 | 利息保障倍数、资产负债率 | 财务指标连续恶化3季淘汰 |
| 第三层：债券条款 | 担保、交叉违约、回售权 | 条款对投资人保护力度 | 条款明显不利于投资人规避 |

## 二、产业债vs城投债的信用定价差异

马龙认为产业债和城投债的信用定价逻辑存在本质差异：

| 维度 | 产业债 | 城投债 |
|------|-------|-------|
| 信用锚 | 企业自身现金流 | 地方政府支持意愿 |
| 风险来源 | 经营风险+行业周期 | 政策风险+财政实力 |
| 定价因子 | 行业景气>企业财务>条款 | 区域经济>行政级别>平台地位 |
| 违约后果 | 市场化处置 | 政府协调化解 |
| 马龙偏好 | ★★★★★ 核心仓位 | ★★★ 选择性配置 |

> **马龙核心观点**：产业债的风险更可分析和可预测（基于财务模型和行业研究），而城投债的风险更多依赖政策判断，因此产业债更适合深度研究驱动的投资策略。

## 三、流动性管理心得——招商系平台优势

马龙依托招商基金（招商银行子公司）的平台优势，形成了独特的流动性管理体系：

| 平台优势 | 具体体现 |
|----------|---------|
| 同业网络 | 招商银行广泛的同业合作关系，支持大额融资和债券借贷 |
| 资金定价 | 银行间市场定价权较强，融资成本低于行业平均 |
| 信息优势 | 招商银行信贷客户资源，提供产业债发行人的一手经营数据 |
| 交易能力 | 招商基金固收团队规模超30人，交易覆盖全市场 |

## 四、经典投资案例：某央企电力债重仓逻辑

| 维度 | 分析 |
|------|------|
| 投资标的 | 国家电网中期票据（AAA） |
| 配置比例 | 占净值约4%-5%，长期第一大持仓 |
| 行业景气 | 电力需求刚性增长+电网投资持续+现金流极度稳定 |
| 企业财务 | 资产负债率约55%、利息保障倍数8倍+、经营现金流超4000亿 |
| 债券条款 | 无担保但央企信用实质担保、流动性好、久期匹配组合需求 |
| 持有周期 | 2018年至今（超7年），滚动续作 |
| 核心逻辑 | 现金流极度稳健（类利率债）+票息高于国债80-100bp |

## 五、投资心得总结

1. **产业债的本质是现金流分析**：行业现金流>企业现金流>债券现金流
2. **信用下沉有边界**：不碰强周期行业，不碰高杠杆主体，不碰条款不利的债券
3. **流动性管理是生命线**：招商系平台优势是独有壁垒，杠杆率和融资成本管控是超额收益的重要来源
4. **波段操作增强收益**：利率债波段+信用债票息积累，双轮驱动

## 六、风险提示

- 基金经理个人风格不代表未来表现
- 平台优势可能随行业竞争加剧而弱化
- 信用债投资存在违约风险，过往案例成功不保证未来
