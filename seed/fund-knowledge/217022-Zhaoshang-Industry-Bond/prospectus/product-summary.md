# 招商产业债券A · 基金产品资料概要

## 一、产品概况

| 项目 | 内容 |
|------|------|
| 基金简称 | 招商产业债券A |
| 基金代码 | 217022 |
| 基金类型 | 债券型 · 纯债 · 产业债 |
| 基金管理人 | 招商基金管理有限公司 |
| 风险等级 | R2 · 中低风险 |

## 二、投资目标

在严格控制风险并保持良好流动性的前提下，通过积极主动的投资管理，力争获得高于业绩比较基准的投资回报。

## 三、投资范围

本基金主要投资于债券资产，包括国债、金融债、企业债、公司债、中期票据、短期融资券等。本基金 **不投资于股票**（可转债除外），债券投资比例不低于基金资产净值的 **80%**。

## 四、费率结构

| 费用类型 | 费率 |
|----------|------|
| 管理费 | **0.60%** / 年 |
| 托管费 | **0.20%** / 年 |
| 申购费（前端） | 0.80%（100 万以下示例档） |
| 赎回费 | 持有 7—365 天 0.10%；365 天以上 0.00% |

## 五、风险揭示

本基金为 **债券型基金**，预期风险和预期收益高于货币市场基金，低于股票型基金。信用债投资可能面临发行人违约等信用风险，利率变动可能导致净值波动。
