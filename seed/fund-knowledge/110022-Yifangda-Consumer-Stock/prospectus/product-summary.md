# 易方达消费行业股票 · 基金产品资料概要

## 一、产品概况

| 项目 | 内容 |
|------|------|
| 基金简称 | 易方达消费行业股票 |
| 基金代码 | 110022 |
| 基金类型 | 股票型 · 行业 · 消费 |
| 基金管理人 | 易方达基金管理有限公司 |
| 风险等级 | R4 · 中高风险 |

## 二、投资目标

本基金主要投资于消费行业上市公司股票，在严格控制风险的前提下，追求超越业绩比较基准的投资回报。

## 三、投资范围

本基金投资于具有良好流动性的金融工具，包括国内依法发行上市的股票、债券、货币市场工具及法律法规允许的其他金融工具。股票投资比例不低于基金资产净值的 **80%**，其中投资于消费行业股票的比例不低于非现金基金资产的 **80%**。

## 四、费率结构

| 费用类型 | 费率 |
|----------|------|
| 管理费 | **1.20%** / 年 |
| 托管费 | **0.20%** / 年 |
| 申购费（前端） | 1.50%（100 万以下示例档） |
| 赎回费 | 持有不满 7 天 1.50%；365 天以上 0.00% |

## 五、风险揭示

本基金为 **股票型基金**，预期风险和预期收益高于混合型基金、债券型基金和货币市场基金。消费行业受宏观经济、政策及景气周期影响，净值波动可能较大。
