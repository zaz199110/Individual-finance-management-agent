# 2026年同业存单配置：短久期工具的角色

**观点类型**：固收市场研究摘录（演示种子）  
**适用基金**：017704 兴业中证同业存单 AAA 指数 7 天持有期

## 核心观点

1. 同业存单指数产品适合作为 **现金管理增强** 工具，收益通常略高于货币基金，但需接受 7 天锁定期。
2. 2026 年利率中枢若下行，存单指数产品净值表现相对平稳，波动小于中长债。
3. 与 QDII 权益（如 019305）搭配，可形成 **稳健底仓 + 海外权益** 的简单组合框架。

## 适用场景

| 场景 | 说明 |
|------|------|
| 工资结余 | 7 天后可赎回，适合月度结余短期存放 |
| 待投闲置金 | 等待更好买点前的过渡配置 |
| 组合底仓 | 与权益/QDII 搭配降低整体波动 |

## 风险提示

- 非存款，不承诺保本保收益
- 7 天内不可赎回
- 利率上行阶段收益可能承压
