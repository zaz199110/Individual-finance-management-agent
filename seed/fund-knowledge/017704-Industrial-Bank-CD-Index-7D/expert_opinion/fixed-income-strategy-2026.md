# 短久期固收产品配置策略

**观点类型**：固收策略研究摘录（演示种子）  
**适用基金**：017704 兴业中证同业存单AAA指数7天持有期

## 一、低利率环境下的配置困境

2026年，国内10年期国债收益率持续在 **2.5%—2.8%** 区间震荡，短端利率同步下移。投资者面临"不投贬值、投了怕亏"的困境：

| 工具 | 年化收益（估） | 波动特征 | 锁定期 |
|------|-------------|---------|--------|
| 活期存款 | 0.20% | 无波动 | 无 |
| 货币基金 | 1.30%—1.50% | 极低 | 无（部分T+1） |
| **存单指数基金** | **+0.30%—0.50% vs 货基** | 极低 | 7天 |
| 短债基金 | +0.50%—1.00% vs 货基 | 低—中 | 无/较短 |
| 中长期纯债 | +1.00%—2.00% vs 货基 | 中 | 无 |

## 二、同业存单指数基金的差异化定位

### 核心优势
1. **收益增强**：较货币基金高出约30—50bp年化收益
2. **信用安全**：仅投资AAA级存单，信用风险极低
3. **费率低廉**：管理费0.20%/年，显著低于主动债基
4. **透明度高**：指数化运作，持仓清晰可查

### 局限与适用边界
1. 7天锁定期 → 不适用于随时取用的活钱
2. 收益不保本 → 极端利率上行时可能有微幅回撤
3. 无法替代中长债 → 利率下行周期中，弹性不及长久期品种

## 三、2026年配置建议框架

```
现金管理类 → 货币基金（活钱，0-7天）
    ↓ 持有期 ≥ 7天
短久期底仓 → 同业存单指数基金（稳健增强）
    ↓ 持有期 ≥ 3个月
中短债增强 → 短债基金（适度拉长久期）
    ↓ 
综合配置 → 股债平衡 / 固收+（长期资产增值）
```

## 四、与权益资产的搭配价值

在2026年市场波动加大的背景下，存单指数基金作为**稳定器**的功能凸显：

| 组合方案 | 权益部分 | 固收底仓 | 适用场景 |
|---------|---------|---------|---------|
| 保守型 | 0—10% | **存单指数 + 短债** | 稳健保值 |
| 均衡型 | 30—50% | **存单指数（底仓）** | 攻守兼备 |
| 进取型 | 70—90% | **存单指数（流动性储备）** | 等待买点的过渡资金 |

## 五、风险提示

- 利率快速上行可能导致存单指数基金净值短暂回撤（幅度通常<1%）
- 极端流动性冲击下，银行存单市场可能出现阶段性冻结
- 非存款类产品，不享受存款保险保障
