# 兴业中证同业存单AAA指数7天持有期证券投资基金 · 基金产品资料概要

编制说明：本文件为演示知识库种子，内容依据兴业基金管理有限公司官网及公开基金档案整理。

## 一、产品概况

| 项目 | 内容 |
|------|------|
| 基金简称 | 兴业中证同业存单AAA指数7天持有期 |
| 基金代码 | 017704 |
| 基金类型 | 指数型-固收 |
| 基金管理人 | 兴业基金管理有限公司 |
| 基金托管人 | 交通银行股份有限公司 |
| 成立日期 | 2023年3月1日 |
| 运作方式 | 开放式（普通开放式） |
| 最短持有期 | **7天**（每份基金份额设置7天最短持有期） |

## 二、投资目标

本基金通过指数化投资，争取在扣除各项费用之前获得与标的指数相似的总回报，追求跟踪偏离度及跟踪误差的最小化。

## 三、投资范围

本基金主要投资于标的指数成份券和备选成份券。为更好地实现基金的投资目标，本基金还可以投资于：

- 非标的指数成份券和备选成份券的其他同业存单
- 国债、金融债、企业债、公司债、央行票据、地方政府债
- 短期融资券（含超短期融资券）、中期票据
- 资产支持证券、债券回购、银行存款、现金等货币市场工具

**投资限制：**

- 本基金**不投资于股票**
- 本基金**不投资于**可转换债券（可分离交易可转债的纯债部分除外）、可交换债券及其他带有权益属性的金融工具

## 四、业绩比较基准

中证同业存单AAA指数收益率 × 95% + 银行人民币一年定期存款利率（税后）× 5%

## 五、风险收益特征

本基金为债券型基金，其预期风险和预期收益高于货币市场基金，低于混合型基金和股票型基金。本基金主要投资于同业存单，属于中低风险品种。

| 风险等级 | 说明 |
|----------|------|
| 典型风险等级 | **R2 中低风险**（以销售机构评定为准） |
| 主要风险 | 利率风险、信用风险、流动性风险、指数跟踪误差 |

## 六、费率结构

| 费用类型 | 费率 |
|----------|------|
| 管理费 | **0.20%** / 年 |
| 托管费 | **0.05%** / 年 |
| 销售服务费（C类） | **0.20%** / 年 |
| 申购费 | **0.00%**（前端） |
| 赎回费 | **0.00%**（满7天持有期后） |

## 七、赎回与持有期规则

本基金对每份基金份额设置**7天的最短持有期**。对于每份基金份额，投资者自其持有的基金份额的锁定到期日的下一工作日起，方可申请赎回或转换转出业务。

**对投资者的含义：**

- 买入后 **7 天内不能赎回**
- 满 7 天后赎回 **不收取赎回费**
- 适合作为 **短期闲置资金** 的稳健配置工具，而非随时取用的活期存款替代品

## 八、投资策略（摘要）

1. **指数化投资**：主要采取抽样复制或完全复制等方法跟踪中证同业存单AAA指数
2. **久期与券种管理**：在跟踪误差可控前提下，优化组合结构
3. **流动性管理**：保持适当现金及流动性资产，满足赎回需求

## 九、适合什么样的投资者？

- 希望获得 **略高于货币基金** 的收益弹性
- 能接受 **7天锁定期**，资金用途在 **1周以上**
- 风险偏好 **稳健**，不愿承担股票波动
- 需要 **现金管理 / 底仓** 类工具的投资者
