# 被动指数基金：费率与跟踪质量如何权衡

**观点类型**：基金研究点评（演示种子）

## 评价维度

| 维度 | 说明 |
|------|------|
| 跟踪误差 | 近一年日均偏离度是否稳定低于合同阈值 |
| 费率拖累 | 管理费+托管费+销售服务费对长期收益的累积影响 |
| 规模与流动性 | 规模过小可能带来清盘风险；规模过大对 QDII 换汇效率有要求 |
| 运营经验 | QDII 额度、换汇时点、指数复制方式 |

## 对 019305 的解读提示

- 指数型 QDII 的核心价值是 **规则透明、费用可预期**
- 写报告时应用 L0 业绩 + L1 披露费率 **交叉验证**，勿仅引用本观点
- 若跟踪误差阶段性放大，应查季报「跟踪误差说明」章节（FK-CITE）
