# 天弘余额宝货币 · 基金产品资料概要

## 一、产品概况

| 项目 | 内容 |
|------|------|
| 基金简称 | 天弘余额宝货币 |
| 基金代码 | 000198 |
| 基金类型 | 货币型 |
| 基金管理人 | 天弘基金管理有限公司 |
| 风险等级 | R1 · 低风险 |

## 二、投资目标

在严格控制投资风险并保持基金资产流动性的基础上，力争获得高于业绩比较基准的稳定收益。

## 三、投资范围

本基金投资于具有良好流动性的货币市场工具，包括现金、期限在一年以内（含一年）的银行存款、债券回购、中央银行票据、同业存单、剩余期限在 397 天以内的债券、资产支持证券等。**本基金不投资股票**，投资组合平均剩余期限不超过 120 天。

## 四、费率结构

| 费用类型 | 费率 |
|----------|------|
| 管理费 | **0.15%** / 年 |
| 托管费 | **0.05%** / 年 |
| 销售服务费 | **0.25%** / 年 |
| 申购费 | **0.00%** |
| 赎回费 | **0.00%** |

## 五、风险揭示

本基金为 **货币市场基金**，属于低风险品种，预期风险和预期收益低于债券型基金、混合型基金和股票型基金。但不保证在每一交易日实现盈利，也不保证最低收益。
