## 五、重要货币基金监管合规情况

《重要货币市场基金监管暂行规定》于2025年正式实施，余额宝作为首批认定的重要货币市场基金，下半年持续满足以下合规要求：

| 监管指标 | 监管要求 | 实际水平 | 合规状态 |
|---------|---------|---------|---------|
| 平均剩余期限 | ≤120天 | 73-78天 | ✓ 合规 |
| 流动性资产占比 | ≥30% | 70-73% | ✓ 合规 |
| 单一银行存款集中度 | ≤10% | <5% | ✓ 合规 |
| 风险准备金 | 200亿元以上 | 满足 | ✓ 合规 |
| 净资产规模约束 | 20,000亿上限 | ~7,500亿 | ✓ 合规 |

## 六、利率风险管理

下半年利率风险管理重点：
- **久期控制**：维持平均剩余期限在73-78天，低于120天监管上限
- **信用风险**：同业存单全部配置于AAA级，银行存款分散于全国性商业银行
- **流动性储备**：维持LCR在275%-295%区间，远超100%监管要求

## 七、基金经理运作说明

基金经理团队下半年运作稳健，未发生基金经理变更。在低利率环境下坚持以流动性安全为底线，通过精细化管理提升组合收益。
