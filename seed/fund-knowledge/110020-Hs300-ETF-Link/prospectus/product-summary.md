# 易方达沪深300ETF联接A · 基金产品资料概要

编制说明：本文件为演示知识库种子，内容依据易方达基金管理有限公司公开信息整理。产品原型：**C 类 · 宽基指数联接**。

## 一、产品概况

| 项目 | 内容 |
|------|------|
| 基金简称 | 易方达沪深300ETF联接A |
| 基金代码 | 110020 |
| 基金类型 | 指数型 · 股票 · ETF 联接 |
| 基金管理人 | 易方达基金管理有限公司 |
| 基金托管人 | 中国建设银行股份有限公司 |
| 成立日期 | 2011年8月29日 |
| 运作方式 | 开放式 |
| 跟踪标的 | 沪深300指数 |

## 二、投资目标

通过投资于目标 ETF，紧密跟踪标的指数，追求跟踪偏离度和跟踪误差的最小化。

## 三、投资范围

本基金主要投资于目标 ETF（易方达沪深300ETF），以及标的指数成份券和备选成份券。为更好地实现投资目标，本基金还可投资于：

- 非标的指数成份券的其他股票
- 债券、货币市场工具及法律法规允许的其他金融工具

**投资限制（摘要）**：

- 投资于目标 ETF 的比例不低于基金资产净值的 **90%**
- 投资于标的指数成份券和备选成份券的比例合计不低于基金资产净值的 **90%**
- 现金或到期日在一年以内的政府债券不低于基金资产净值的 **5%**

## 四、业绩比较基准

沪深300指数收益率 × 95% + 银行人民币活期存款利率（税后）× 5%

## 五、费率结构

| 费用类型 | 费率 |
|----------|------|
| 管理费 | **0.15%** / 年 |
| 托管费 | **0.05%** / 年 |
| 申购费（前端） | 0.12%（100 万以下示例档） |
| 赎回费 | 持有不满 7 天 1.50%；7 天—365 天 0.50%；365 天以上 0.00% |

## 六、风险揭示

本基金为 **股票型指数基金**，其预期风险和预期收益高于混合型基金、债券型基金和货币市场基金。

| 风险类型 | 揭示要点 |
|----------|----------|
| 市场风险 | 标的指数及成份股价格波动可能导致基金净值大幅波动 |
| 跟踪误差风险 | 联接基金运作、费用及抽样复制等因素可能导致与标的指数表现存在偏差 |
| 流动性风险 | 成份股或目标 ETF 流动性不足时，可能影响申赎与跟踪效果 |
| 典型风险等级 | **R4 中高风险**（以销售机构评定为准） |

## 七、分红政策

在符合分红条件时，基金管理人可根据基金运作情况分配收益；是否分红、分红次数与比例不保证。
